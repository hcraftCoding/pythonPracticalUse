# Python API Prep

Foundational exercises building toward real-world Python API workflows.
Each exercise isolates a core skill that every production API integration requires.

## Why this exists

Before writing my first API call, I'm deliberately building the underlying
skills — data parsing, error handling, and file I/O — so the integration
work is clean when I get there.

## Exercises

| File | Skill | Real-world connection |
|------|-------|----------------------|
| `exercise_01_dict_parsing.py` | Dictionary and JSON navigation | Parsing API responses |
| `exercise_02_functions_error_handling.py` | Functions and try/except | Handling failed API calls gracefully |
| `exercise_03_file_io.py` | File I/O and the json module | Storing and loading API data |

## Stack
- Python 3
- Standard library only (no external packages yet)
